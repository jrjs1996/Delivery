# Settings Menu

Updated 18/6/2019

Displays a menu that is populated with the menu items provided in
children. When one of the children is clicked on the onClick function
will be called with the index of the child. All children should be
SettingsMenuItems.

<Props of={SettingsMenu} />

<Playground>
  <span id="showData"></span>
  <SettingsMenu
    title="Choose your language"
    onClick={i => {
      let message = `You chose item at index ${i}. `;
      switch (i) {
        case 0:
          message += "Hello!";
          break;
        case 1:
          message += "Bonjour!";
          break;
      }
      document.getElementById("showData").innerHTML = message;
    }}>
    <SettingsMenuItem text="English" />

  </SettingsMenu>
