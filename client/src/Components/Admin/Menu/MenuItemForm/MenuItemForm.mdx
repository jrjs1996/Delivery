# Admin Menu Item Form

Updated 20/06/2019

Form for editing menu items. Can either add a new menu item or update
an existing one based on whether or not the menuItem prop is provided.

<Props of={MenuItemForm} />

<Playground>
  {() => {
    const [message, setMessage] = useState('');

    return (
      <div>
        <p>{ message }</p>
        <MenuItemForm
          addMenuItem={i => setMessage('Add Item: ' + JSON.stringify(i))}
          updateMenuItem={i => setMessage('Update Item: ' + JSON.stringify(i))}
        />
      </div>
    );
  }}
</Playground>
