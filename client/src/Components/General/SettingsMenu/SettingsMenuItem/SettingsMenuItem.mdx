# Settings Menu Item

Updated 18/6/2019

Used as a child of SettingsMenu. Represents a single
item in the menu.

  <SettingsMenu
    title="Choose your language"
    onClick={i => {
      let message = `You chose item at index ${i}. `;
      switch (i) {
        case 0:
          message += "Hello!";
          break;
        case 1:
          message += "Bonjour!";
          break;
      }
      document.getElementById("showData").innerHTML = message;
    }}>
