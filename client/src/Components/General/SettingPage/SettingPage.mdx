# Setting Page

Updated: 17/6/2019

This component is used to create a settings page. It takes SettingPageInputs
as children. The onSubmit function will be called when the submit button is
clicked.

The function will receive on object with properties and values from
each SettingPageInput in children. e.g.

{ child1Name: "child1Value", child2Name: "child2Value" ...}

<Props of={SettingPage} />

<Playground>
  <span id="showData"></span>
  <SettingPage
    title="Example"
    onSubmit={submitData => {document.getElementById("showData").innerHTML = JSON.stringify(submitData)}}
  >
    <SettingPageInput
      fullWidth
      label="Name"
      name="name"
    />
    <SettingPageInput
      fullWidth
      label="Age"
      name="age"
    />
  </SettingPage>
</Playground>
